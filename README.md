# You-music

```tex
本项目使用React、Typescript，Css-in-js搭建
```

### Demo

```tex
http://you-music.top/
```

### 安装依赖

```tex
npm install

yarn install
```

### 构建

```tex
npm build

yarn build
```

### 运行

```tex
npm start

yarn start
```

### 功能

```tex
·歌词随播放进度滚动
·单曲、随机、顺序，列表循环
·音量调节
·移动端基础适配
·每日推荐歌曲
·推荐歌单
·歌曲评论
·调整播放进度
·网易云音乐App扫码登录
·排行榜
·我喜欢的音乐(需要登录)
·最近播放(需要登录)
·双击播放
```

### 灵感来自

```tex
中国色
http://zhongguose.com/

网易云开源接口
https://github.com/Binaryify/NeteaseCloudMusicApi
```
